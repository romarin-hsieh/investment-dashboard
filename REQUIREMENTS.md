# 投資儀表板需求規格書 (Requirements Specification)

## 📋 專案概述

**專案名稱**: Investment Dashboard POC  
**版本**: v2.0  
**最後更新**: 2025-12-24  
**狀態**: 生產就緒 (Production Ready)

### 專案目標
建立一個現代化的投資信息儀表板，整合市場數據、股票分析和恐懼貪婪指數，提供投資者全面的市場洞察。

### 核心價值主張
- **即時市場數據**: 整合 TradingView 提供實時市場信息
- **智能分析**: 結合技術分析和基本面數據
- **用戶友好**: 響應式設計，支持多設備訪問
- **高性能**: 優化的載入策略和緩存機制

---

## 🎯 功能需求

### 1. 市場儀表板 (Market Dashboard)

#### 1.1 主要市場指數
- **顯示內容**: S&P 500, NASDAQ, Russell 2000, Dow Jones, Gold, Oil
- **數據來源**: TradingView Tickers Widget
- **更新頻率**: 實時
- **顯示格式**: 橫向滾動條，包含價格和變化百分比

#### 1.2 VIX 恐慌指數
- **VIX 指標**: 快速概覽儀表盤
- **VIX 圖表**: 詳細的歷史趨勢圖
- **數據來源**: TradingView VIX Widget

#### 1.3 Fear & Greed Index
- **實現方式**: Zeiierman 7 組件算法
- **顯示格式**: 半圓形儀表盤 + 組件分解
- **設計風格**: CNN 風格的低飽和度配色
- **當前數值**: 58 (Greed)

#### 1.4 Top Stories 新聞
- **數據來源**: TradingView Timeline Widget
- **顯示模式**: 市場相關新聞
- **容器高度**: 650px
- **主題**: 淺色主題，透明背景

#### 1.5 Stock Market Insight
- **Daily 分析**: NASDAQ 和 US30 的日線圖表
- **Weekly 分析**: NASDAQ 和 US30 的週線圖表
- **排列方式**: 垂直排列 (上下結構)
- **容器高度**: 每個 600px

#### 1.6 股票熱力圖
- **數據來源**: TradingView Stock Heatmap
- **數據集**: S&P 500, NASDAQ 100 等
- **容器高度**: 500px
- **控制選項**: 已移除選擇器

### 2. 股票儀表板 (Stock Dashboard)

#### 2.1 股票概覽
- **符號配置**: 支持環境變數和 Google Sheets 動態配置
- **緩存機制**: 10 分鐘緩存間隔
- **分組邏輯**: 按 Yahoo Finance 11 個標準行業分類
- **信心閾值**: >= 0.7 可接受，< 0.7 歸類為 Unknown
- **排序規則**: 行業內按符號 A-Z 排序

#### 2.2 股票卡片 (Stock Cards)
- **佈局比例**: Symbol Overview (2/3) : Technical Analysis (1/3)
- **容器高度**: 440px (優化後)
- **行業標籤**: 彩色編碼的行業分類標籤
- **Detail 按鈕**: 導航到詳細分析頁面

#### 2.3 Ticker Tape
- **位置**: 股票標題和行業分組之間
- **符號來源**: 基礎市場指標 + 配置的股票符號
- **容器高度**: 82px
- **邊框**: 已移除邊框線

### 3. 股票詳細頁面 (Stock Detail)

#### 3.1 頁面結構
- **第一行**: Symbol Overview (2/3) + Technical Analysis (1/3)
- **第二行**: Company Profile (全寬)
- **第三行**: Fundamental Data (全寬)

#### 3.2 導航功能
- **麵包屑**: Stock Dashboard > 股票分析
- **返回按鈕**: ← Back to Stock Dashboard
- **行業標籤**: 與 Stock Dashboard 一致的顏色編碼

#### 3.3 Widget 高度優化
- **Symbol Overview/Technical**: 440px
- **Company Profile**: 400px
- **Fundamental Data**: 600px

### 4. 設定頁面 (Settings)
- **Performance Test**: 性能測試和監控工具
- **Cache Test**: 符號配置緩存測試
- **其他開發工具**: 根據需要擴展

## 🛠️ 技術需求

### 1. 前端技術棧
- **框架**: Vue 3 + Composition API
- **構建工具**: Vite 5.x
- **路由**: Vue Router 4 (Hash 模式)
- **樣式**: 原生 CSS3 + 響應式設計
- **TypeScript**: 部分組件使用 TS

### 2. 數據管理
- **DataFetcher**: 統一的數據獲取層
- **StateManager**: 本地狀態管理
- **緩存策略**: localStorage + 10 分鐘緩存
- **Fallback 機制**: 網路 → 緩存 → 靜態數據

### 3. 性能優化
- **FastTradingViewWidget**: 高性能 Widget 組件
- **Widget 緩存**: 避免重複載入
- **節流控制**: 防止過度請求
- **預載入**: 智能預載入機制

### 4. 符號配置管理
- **多層 Fallback**: 環境變數 → 緩存 → Google Sheets → 靜態
- **緩存時間**: 10 分鐘可配置
- **動態更新**: 支持 Google Sheets 整合
