Manual Testing Guide

⚠️ Manual Testing Required
The following tests require manual verification in the actual application:

Desktop Testing (≥1024px):

  1. Navigate to a stock detail page with MFI Volume Profile widget
  2. Verify chart shows: left side K-line chart, right side Canvas volume profile
  3. Check that all volume bars are visible and readable
  4. Hover over bars to see tooltips with complete information
  5. Verify POC (blue markers) and Value Area (orange background) are visible
  6. Check price labels on left and volume labels on right

Mobile Testing (<1024px):

  1. Verify layout switches to vertical stacking
  2. Canvas volume profile should still be functional
  3. Labels may be reduced but tooltips should provide full info

Feature Flag Testing:

  1. Set useCanvasVolumeProfile=false in widget props
  2. Verify it falls back to original histogram display
  3. Set mfiAvgMode='legacy' to test old averaging method

Code Examples

Using Volume-Weighted MFI (Default):

// Default behavior - volume-weighted averaging const profile = calculateMFIVolumeProfile(ohlcvData, 20, 14); // or explicitly const profile = calculateMFIVolumeProfile(ohlcvData, 20, 14, { mfiAvgMode: 'weighted' });

Using Legacy Count-Based MFI:

// Legacy behavior for rollback const profile = calculateMFIVolumeProfile(ohlcvData, 20, 14, { mfiAvgMode: 'legacy' });

Widget with Canvas (Default):

<MFIVolumeProfileWidget :symbol="symbol" :useCanvasVolumeProfile="true" :mfiAvgMode="'weighted'" />

Widget with Histogram Fallback:

<MFIVolumeProfileWidget :symbol="symbol" :useCanvasVolumeProfile="false" :mfiAvgMode="'legacy'" />
