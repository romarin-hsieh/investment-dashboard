🧪 測試指南

視覺測試步驟：

  1. 啟動開發伺服器：
    npm run dev
  2. 導航到 Stock Overview：
    http://localhost:5173/#/stock-overview
  3. 驗證預設展開：
    • 首次載入時所有 sector 和 industry 應該都是展開的
    • 可以看到所有股票符號
  4. 驗證階層縮排：
    • Sector 標題應該在最左邊
    • Industry 標題應該有適度縮排
    • Symbol 應該有最多的縮排
  5. 驗證文字溢位：
    • 縮小瀏覽器視窗寬度
    • 長文字應該顯示 "..." 省略號
    • 滑鼠懸停應該顯示完整文字 tooltip
  6. 驗證無邊框設計：
    • 所有 node 應該沒有邊框
    • 背景應該是透明的
    • hover 效果應該只有背景色變化
