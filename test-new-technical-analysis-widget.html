🔧 創建專用的 TechnicalAnalysisWidget 組件

基於你提供的原生 TradingView HTML 結構，創建了一個新的專用組件：

  • 保留 FastTradingViewWidget 的所有優點：載入效果、錯誤處理、優先級載入
  • 使用原生 TradingView 結構：完全按照你提供的 HTML 結構實現
  • 完美的樣式一致性：使用 overview-widget 樣式，確保與 Symbol Overview 一致
  • 乾淨的代碼分離：Technical Analysis 有自己的組件，不影響其他功能
🏗️ 技術實現詳情

1. 新建 TechnicalAnalysisWidget.vue

<template> <div class="fast-widget overview-widget"> <!-- 載入狀態 --> <div v-if="!loaded && !error" class="fast-loading"> <div class="loading-spinner"></div> <span>Loading Technical Analysis...</span> </div> <!-- 原生 TradingView 結構 --> <div v-show="loaded" class="tradingview-widget-container"> <div class="tradingview-widget-container__widget"></div> <div class="tradingview-widget-copyright"> <a href="...">{{ symbol }} stock analysis</a> <span class="trademark"> by TradingView</span> </div> </div> </div> </template>

2. 更新 StockCard.vue

<!-- 使用新的 TechnicalAnalysisWidget --> <div class="widget-overview"> <div class="widget-header"> <h4>Technical Analysis</h4> </div> <TechnicalAnalysisWidget :symbol="quote.symbol" :exchange="getExchange()" :priority="2" /> </div>

3. 配置與你提供的範例完全一致

getWidgetConfig() { return { "colorTheme": "light", "displayMode": "single", "isTransparent": true, "locale": "en", "interval": "1D", "disableInterval": false, "width": "100%", "height": "100%", "symbol": `${this.exchange}:${this.symbol}`, "showIntervalTabs": true } }
