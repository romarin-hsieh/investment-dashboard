🔧 Symbol Insight Lazy Widget Padding Fix

🎯 修改目標

📝 調整 Symbol Insight 區塊的 lazy-widget padding

將 Daily Insight (MA5) 和 Weekly Insight (MA4) 的 lazy-widget padding 設為 8px：

  • 目標元素：`.insight-widget .lazy-widget`
  • 修改內容：添加 `padding: 8px` 和 `box-sizing: border-box`
  • 影響範圍：僅影響 Symbol Insight 區塊內的 LazyTradingViewWidget
  • 目的：與其他 widget 保持一致的內邊距設計
🔧 CSS 修改詳情

在 StockDetail.vue 中添加的 CSS 規則

/* Symbol Insight 區塊內的 lazy-widget 添加 8px padding */ .insight-widget .lazy-widget { padding: 8px; box-sizing: border-box; }

選擇器說明

  • .insight-widget：Symbol Insight 區塊內的 widget 容器
  • .lazy-widget：LazyTradingViewWidget 組件的根元素
  • 組合效果：只影響 Symbol Insight 區塊內的 LazyTradingViewWidget
  • 不影響：其他頁面或區塊中的 LazyTradingViewWidget
