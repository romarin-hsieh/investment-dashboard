🏢 Company Profile Symbol Fix Test

Testing the Company Profile widget to ensure it displays the correct symbol for each stock detail page

🎯 Problem Fixed COMPLETED
The Company Profile widget was hardcoded to show "RDW Profile" regardless of the current stock symbol. This has been fixed to use the correct symbol and exchange.

❌ Before (Problem)

Hardcoded symbol in config:

const config = { "symbol": "NYSE:RDW", // 硬編碼 "colorTheme": "light", "isTransparent": true, "locale": "en", "width": "100%", "height": "100%" }

Issue: 所有股票詳情頁面都顯示 RDW 的公司資料

✅ After (Fixed)

Dynamic symbol from props:

const config = { "symbol": this.fullSymbol, // 動態使用 props "colorTheme": this.colorTheme, "isTransparent": this.isTransparent, "locale": this.locale, "width": "100%", "height": "100%" }

Result: 每個股票頁面顯示對應的公司資料

📁 Modified Files
The following component has been updated to use dynamic symbol configuration:
  • ✅ src/components/TradingViewCompanyProfile.vue - Fixed hardcoded symbol issue
  • ✅ src/pages/StockDetail.vue - Removed duplicate import
