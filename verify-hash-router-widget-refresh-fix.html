Test Case 2: Hash Router Query Format Consistency

Objective: Verify focus parameters use correct hash router format

Step 1: Navigate to Stock Overview

Go to the stock overview page:

https://your-domain.github.io/investment-dashboard/#/stock-overview

Step 2: Click on Any Symbol

Click on any stock symbol in the navigation panel or stock cards

Expected Result: Correct URL Format

URL should update to correct hash router format:

✅ Correct: #/stock-overview?focus=SYMBOL
❌ Incorrect: ?focus=SYMBOL#/stock-overview
  • Focus parameter appears after the hash and route path
  • No focus parameter in main URL search params
  • Page scrolls to the selected symbol
