🔧 Technical Analysis Padding Fix - 消除雙重框框

🎯 問題描述

🔍 雙重框框問題

Technical Analysis 區塊出現了兩個框框（雙重邊框效果）：

  • 外層框框：來自 `.widget-technical` 容器的邊框
  • 內層框框：來自 `.technical-overview-style` 的 8px padding + 白色背景
  • 視覺問題：看起來像是 widget 內部又有一個獨立的白色區塊
🔧 解決方案

修改 .technical-overview-style 的 padding

/* 修改前 */ .technical-overview-style { background: #ffffff !important; padding: 8px !important; /* 造成雙重框框 */ box-sizing: border-box !important; } /* 修改後 */ .technical-overview-style { background: #ffffff !important; padding: 0px !important; /* 消除雙重框框效果 */ box-sizing: border-box !important; }

同時調整 loading 和 error 狀態

/* Technical Analysis loading 狀態調整 */ .technical-overview-style .fast-loading { background: #ffffff; border-radius: 0; margin: 0; /* 不再需要抵消 padding */ } /* Technical Analysis error 狀態調整 */ .technical-overview-style .fast-error { background: #ffffff; border: 1px solid #e0e0e0; border-radius: 0; color: #6c757d; margin: 0; /* 不再需要抵消 padding */ }
