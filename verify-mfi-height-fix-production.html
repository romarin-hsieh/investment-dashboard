🧪 開發者工具檢查步驟

  1. 打開任一測試連結 (建議 AAPL)
  2. 滾動到 "MFI Volume Profile" 區塊
  3. 按 F12 打開開發者工具
  4. 在 Elements 面板中找到 MFI Volume Profile 容器
  5. 檢查以下 CSS 屬性：
/* 外層容器應該是自動高度 */ .insight-full-widget.mfi-volume-profile { min-height: auto; height: auto; } /* Chart 容器應該是 600px */ .chart-container { height: 600px; } /* Volume bars 應該包含 50 個 bins */ .volume-bars .volume-bar-row { /* 應該有 50 個這樣的元素 */ height: 12px; }
