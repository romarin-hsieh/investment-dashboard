🔧 TradingView iframe Height Fix Test

Testing the iframe height adjustment for Market Regime and Trading Strategy widgets

🎯 Fix Applied - Version 2 UPDATED
Fixed CSS scoping issue by moving iframe height override to global styles in both Market Regime and Trading Strategy widgets.

❌ Previous Issue

CSS scoped styles couldn't affect dynamically created iframe:

<style scoped> .tradingview-widget-container iframe { height: 100% !important; /* 無法作用 */ } </style>

Problem: Vue scoped styles 無法影響動態創建的 TradingView iframe

✅ Fixed Solution

使用全域樣式搭配特定選擇器:

<style> /* 全域樣式 - 可以影響動態 iframe */ .advanced-chart-widget .tradingview-widget-container iframe { height: 100% !important; } </style>

Result: iframe 現在會正確填滿整個 850px 容器高度
