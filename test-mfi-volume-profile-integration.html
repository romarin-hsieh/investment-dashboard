🏗️ Component Architecture
Data Flow:
1. PrecomputedOhlcvApi.getOhlcv() → Try precomputed data first
2. YahooFinanceAPI.getOhlcv() → Fallback to live Yahoo Finance data
3. calculateMFIVolumeProfile() → Process OHLCV data with MFI analysis
4. getMFIVolumeProfileSignals() → Generate trading signals
5. LightweightCharts → Render K-line chart with volume profile overlay

🎯 Key Features Implemented
MFI Calculation:
• Money Flow Index with configurable period (default: 14)
• Typical Price = (High + Low + Close) / 3
• Raw Money Flow = Typical Price × Volume
• Positive/Negative Money Flow classification
• MFI = 100 - (100 / (1 + Money Flow Ratio))
Volume Profile Analysis:
• Configurable price bins (default: 20)
• Volume distribution across price levels
• MFI-based volume classification (Buying/Selling/Neutral)
• Point of Control (POC) identification
• Value Area calculation (70% of volume around POC)
Trading Signals:
• Market sentiment analysis (BULLISH/BEARISH/NEUTRAL)
• Support and resistance level identification
• Confidence scoring based on volume patterns
• Trading recommendations with context
Chart Visualization:
• Lightweight Charts K-line candlestick display
• Volume profile histogram overlay
• POC and Value Area lines
• Color-coded volume bars (Green=Buying, Red=Selling, Gray=Neutral)
