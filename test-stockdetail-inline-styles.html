🚨 StockDetail 內聯樣式強制修正

⚡ 緊急修正策略

由於 CSS 樣式持續被覆蓋，現在採用內聯樣式 (inline styles) 的方法， 直接在 HTML 元素上使用 style="..." 屬性，並加上 !important 聲明， 這是 CSS 優先級最高的方式，應該能夠強制顯示所有標題和正確的背景顏色。

🔧 修正內容

  • Symbol Overview 標題: 使用內聯樣式強制顯示灰色背景和標題文字
  • Technical Analysis 標題: 使用內聯樣式強制顯示灰色背景和標題文字
  • Daily Insight 標題: 使用內聯樣式強制顯示透明背景和標題文字
  • Weekly Insight 標題: 使用內聯樣式強制顯示透明背景和標題文字
  • 所有主要 Widget: 使用內聯樣式強制白色背景和標題顯示
範例內聯樣式: <div class="widget-header" style="background: #f8f9fa !important; padding: 0.75rem 1rem !important; display: flex !important;"> <h4 style="color: #495057 !important; margin: 0 !important;">Symbol Overview</h4> </div>

💡 如果這次還是不行...

如果內聯樣式都無法解決問題，那可能是：
1. 瀏覽器快取問題 - 請嘗試硬重新整理 (Ctrl+Shift+R)
2. Vue 組件沒有正確重新載入 - 請重新啟動開發伺服器
3. 有其他 JavaScript 在動態修改樣式
