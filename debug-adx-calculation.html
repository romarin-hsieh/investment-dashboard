ADX Formula Verification

ADX Calculation Steps: 1. Calculate True Range (TR) = max(H-L, |H-C₋₁|, |L-C₋₁|) 2. Calculate +DM = max(H-H₋₁, 0) if H-H₋₁ > L₋₁-L, else 0 3. Calculate -DM = max(L₋₁-L, 0) if L₋₁-L > H-H₋₁, else 0 4. Smooth TR, +DM, -DM using Wilder smoothing (period 14) 5. Calculate +DI = 100 × (smoothed +DM / smoothed TR) 6. Calculate -DI = 100 × (smoothed -DM / smoothed TR) 7. Calculate DX = 100 × |+DI - -DI| / (+DI + -DI) 8. Smooth DX using Wilder smoothing (period 14) to get ADX
