🔍 TOC Widget Timeout Debug Tool

問題描述

當使用者點擊 TOC 跳轉到頁面底部時，會觸發大量 Widget 同時載入，導致 timeout 問題。

⚠️ 使用說明：
1. 先開啟 Stock Overview 頁面
2. 開啟瀏覽器 DevTools (F12)
3. 點擊下方測試按鈕
4. 觀察 Console 和 Network 面板

等待開始 Debug...\n
