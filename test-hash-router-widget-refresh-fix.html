Hash Router Widget Refresh Fix - Test Suite

Test 1: Component Remount Behavior

This test verifies that the router-view key implementation forces component remounting on path changes.

Expected Behavior:
  • Path changes (e.g., /symbols/NVDA → /symbols/ONDS) should trigger component remount
  • Query-only changes (e.g., ?focus=A → ?focus=B) should NOT trigger remount
  • Components should be destroyed and recreated on path changes

Test 2: Hash Router Query Format

This test verifies that focus parameters appear in correct hash router format.

Expected Format:
  • Correct: #/stock-overview?focus=SYMBOL
  • Incorrect: ?focus=SYMBOL#/stock-overview

Test 3: NavigationService Deprecation Warnings

This test verifies that deprecated methods show appropriate warnings.

Expected Behavior:
  • Calls to updateQueryParam() should show deprecation warning
  • Calls to getFocusSymbolFromQuery() should show deprecation warning
  • Warnings should suggest using Vue Router query methods

Test 4: URL Parameter Migration

This test verifies handling of legacy URL formats.

Migration Scenarios:
  • Legacy: ?focus=X#/route → Modern: #/route?focus=X
  • Round-trip consistency: parse → serialize → parse should be equivalent
